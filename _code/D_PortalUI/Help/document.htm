Document

The document view displays the document and tools for navigating through the document. It can be accessed via several methods:

  • By clicking on the Document link on the toolbar
  • By opening a title from the Home Page
  • By clicking on a search result

Using the Document Link on the Toolbar

If you have not previously viewed a document, the first time you click on the Document link, the Document Toolbar will appear with an empty document window. To access a document, select the document needed from the Table of Contents and then use the document navagation toolbar to navigate through the document.
If the table of contents is not visible, click the show table of contents button (located on the navagation toolbar) to make it visible. See the section Table of Contents for a detailed description of how to use the table of contents.

Once you have accessed a document from either the table of contents, Home Page, or Search Results, the system will remember the most recent document accessed, and display that document when you click on the Document link on the main toolbar.


Additional Document Toolbar Features

When a document is displayed, these tools appear on the document toolbar:

View/Hide Table of Contents
Use this button to display or hide the Table of Contents pane.
View Source References(FASB ASC Documents Only)
Use this button to view the original pre-codification source reference.
Previous Document
Use this button to move back to the previous document.
Next Document
Use this button to move to the next document.
Print(Document Tab)
Use this button to select printing options.  
Print(All Other Tabs)
This will print all content that is displayed in the tab  
Download Document (available only in certain titles)
Click on the document icon to open the document, or to save the document to your hard drive, right click on the link and choose “Save Target As”. Then choose the location on your hard drive for the exhibit.
Help
This button displays this help document.

When a FASB ASC document is selected, these additional tabs may be displayed:

Go to
This tab  provides the ability Go directly to a topic, subtopic, or section
What Links Here
This tab  provides the ability to see all FASB ASC content that links to a given paragraph in the current document.
Archive
This tab will show the archived versions of the current document.
Cross Reference
This tab presents a cross reference report that identifies where legacy material is now located
Join Sections
This tab allows you to join all of the sections under a given topic.

Document Reference Window

The Reference Window displays the path to the document.  The headings in the path provide a link to that heading.  Click once on the heading to jump to that section of the document.

Join Sections Button

The Join Sections button displays above the table of contents on all subtopic pages. It allows you to combine subdocuments into a single document.
